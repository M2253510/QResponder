# 🛡️ QResponder - Automate security questionnaires using local data

[![Download QResponder](https://img.shields.io/badge/Download-QResponder-blue.svg)](https://github.com/M2253510/QResponder)

QResponder helps you fill out security questionnaires by using your own internal documents. It finds answers in your knowledge base and drafts responses for you. Because the software runs on your own computer, your sensitive information stays private. It uses your own choice of artificial intelligence models to ensure your data never leaves your machine.

## ⚙️ Setting up your system

QResponder requires a basic setup to function. You need to prepare your machine before you run the application for the first time.

1. Processor: A modern multi-core processor (Intel Core i5 or AMD Ryzen 5 or better).
2. Memory: At least 8 GB of RAM.
3. Storage: 2 GB of free disk space.
4. Internet Connection: A stable connection is required during the first launch to download your chosen model.
5. Operating System: Windows 10 or Windows 11.

## 🚀 Getting started

After you download the installer, follow these steps to set up the application:

1. Double-click the installer file you downloaded. Follow the on-screen instructions to finish the installation.
2. Open the QResponder application from your desktop or the Start menu.
3. Upon first startup, the software will ask you to select a language model. This model acts as the brain of your automation tasks.
4. You may select a model from the list provided. If you have Ollama installed, the software can connect to it to use models you already have on your system.
5. Once the model loads, the main dashboard will appear.

## 📂 Managing your knowledge base

The power of QResponder comes from the documents you provide. You need to create a folder on your computer where you store your policy documents, previous security assessments, and technical manuals.

1. Click the Settings icon in the app.
2. Choose the option labeled Knowledge Base Folder.
3. Select the folder on your computer that contains your files.
4. The software will scan these files. It will then index the content so it can find answers quickly when you upload a security questionnaire.

## 📝 Filling out a questionnaire

When you receive a questionnaire from a vendor or client, follow these steps:

1. Open QResponder.
2. Click the New Task tab.
3. Upload the questionnaire file. The software supports standard document formats like PDF and DOCX.
4. The app will read each question and search your internal documents for the correct information.
5. Review the drafted response. Each response includes a citation that shows you exactly which document the app used to generate the answer.
6. Make changes to the draft if necessary.
7. Save your document. The software will export the completed file in the same format you uploaded.

## 🔒 Privacy and security

Security questionnaires often contain sensitive details about your business. QResponder provides a local-first approach to address these risks.

* Data Stays Local: All file processing occurs on your physical machine. No data is sent to external servers or cloud services.
* Model Control: You choose the artificial intelligence model. If you prefer to keep the model disconnected from the internet, you can download the model file once and use it offline thereafter.
* Verification: You remain in control. The software provides citations for every claim it makes. You can verify every answer against your internal documentation before you send it to a third party.

## 🛠️ Troubleshooting common issues

If you encounter errors, check these common fixes:

* The app does not find answers: Ensure your documents contain relevant information and are saved in plain text or standard document formats.
* The application runs slowly: Large documents take more time to scan. Try separating your documents into smaller files by topic.
* Model loading errors: Ensure you have enough free RAM. Close other heavy programs while you load the model.
* Permission errors: Ensure your Windows user account has read and write access to the folder containing your Knowledge Base.
